$$\text{Maximize } 3x_1 + x_2 + 3x_3$$

$$\text{subject to}$$

$$2x_1 + x_2 + x_3 \leq 2$$

$$x_1 + 2x_2 + 3x_3 \leq 5$$

- b. 0
- c. 10

- e. Infactible (UNFEASIBLE)
