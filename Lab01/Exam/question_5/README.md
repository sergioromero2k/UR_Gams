# Find the optimal value of the following problem:

$$\text{minimize } -2x_1 + 4x_2 + 7x_3 + x_4 + 5x_5$$

$$\text{subject to } -x_1 + x_2 + 2x_3 + x_4 + 2x_5 = 7$$

$$-x_1 + 2x_2 + 3x_3 + x_4 + x_5 = 6$$

$$-x_1 + x_2 + x_3 + 2x_4 + x_5 = 4$$

$$x_1 \text{ free}, \quad x_2 \geq 0, \quad x_3 \geq 0, \quad x_4 \geq 0, \quad x_5 \geq 0$$

Select one:

- a. 7
- b. 10
- c. 20
- d. 19
- e. 0
- f. 6
