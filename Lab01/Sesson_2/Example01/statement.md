**Solved Example 01**  
Find the optimum of the following linear programming problem:

$$\text{min} \quad 3x_1 - 5x_2 + x_3$$

$$\text{s.t.} \quad x_1 - x_3 = 1$$

$$3x_1 + 2x_2 - 7x_3 = 5$$

$$x_1, \quad x_2, \quad x_3 \geq 0$$
