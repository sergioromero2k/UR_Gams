# Solved Example 06
Find the optimum of the following linear programming problem:

$$\text{max} \quad 8x_1 + 3x_2 - 2x_3$$

$$\text{s.t.} \quad x_1 - 6x_2 + x_3 \geq 2$$

$$5x_1 + 7x_2 - 2x_3 = -4$$

$$x_1 \leq 0$$

$$x_2 \geq 0$$

$$x_3 \quad \text{unrestricted}$$

**Solution:**  
$x_1 = 0$  
$x_2 = 0$  
$x_3 = 2$ $\qquad Z_{\text{MAX}} = -4$
