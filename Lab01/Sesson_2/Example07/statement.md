# Solved Example 07
Find the optimum of the following problem:

$$\text{max} \quad 6x_1 + 5x_2 + 9x_3 + 40x_4$$

$$\text{s.t.} \quad x_1 + x_3 + 4x_4 \leq 3$$

$$x_2 + x_3 + 5x_4 \leq 4$$

$$x_1 \leq 0, \quad x_2 \leq 0, \quad x_3 \leq 0, \quad x_4 \geq 0$$

**Solution:**  
$x_1 = 0$  
$x_2 = 0$  
$x_3 = -1$  
$x_4 = 2$ $\qquad Z_{\text{MAX}} = 31$
