# Exercise 2
Find the optimum of the following problem:

$$\text{max} \quad 6x_{11} + 2x_{12} + 8x_{13} + 5x_{14} + 9x_{21} + 3x_{22} + 5x_{23} + 8x_{24} +$$

$$4x_{31} + 8x_{32} + 3x_{33} + 4x_{34} + 6x_{41} + 7x_{42} + 6x_{43} + 4x_{44}$$

$$\text{s.t.} \quad \sum_{i=1}^{4} x_{ij} = 1, \qquad \forall j = 1, \ldots, 4$$

$$\sum_{j=1}^{4} x_{ij} = 1, \qquad \forall i = 1, \ldots, 4$$

$$x_{ij} \in \{0, 1\}, \qquad \forall i = 1, \ldots, 4, \; \forall j = 1, \ldots, 4$$
